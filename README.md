# Saltslack

Saltslack is a slack bot (don't ask why I'm not sure myself) that allows users to track repos and recieve automated messages about commits. Simply using /track https://github.com/repousername/reponame or /trackrepo repousername/reponame, users in DMs will recieve automated messages every 5 minutes if a commit was made since the last 5 minutes. You can also use /updaterepos and /untrackrepo
